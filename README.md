## How work?

* Install Node.js [http://https://nodejs.org/]
* Run in terminal: `npm install -g ember-cli`
* `git clone https://github.com/danielpintscher/wflistcards.git`
* Go to the Folder: `cd wflistcards` 
* Run: `npm install`
* Run: `bower install`
* Have fun: `ember s`
